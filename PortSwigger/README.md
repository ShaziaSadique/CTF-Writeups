# PortSwigger Web Security Academy Labs

A collection of my solutions and notes for the **PortSwigger Web Security Academy** labs. This repository documents my hands-on learning in web application security through practical exploitation of real-world vulnerabilities.

> **Disclaimer:** These write-ups are for educational purposes only. All testing was performed within the official PortSwigger Web Security Academy lab environment.

---

## About

The goal of this repository is to:

- Document completed PortSwigger labs
- Explain vulnerabilities in a simple way
- Share step-by-step exploitation methods
- Record payloads and testing techniques
- Include screenshots where applicable
- Build a personal knowledge base for web penetration testing

---

## Lab Categories

- SQL Injection (SQLi)
- Cross-Site Scripting (XSS)
- Cross-Site Request Forgery (CSRF)
- Authentication
- Access Control
- File Path Traversal
- File Upload Vulnerabilities
- Command Injection
- Server-Side Request Forgery (SSRF)

> More lab categories will be added as I continue completing the Web Security Academy.
